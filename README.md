# Tesla Clone

## Tesla Clone with ReactJS (Redux + Styled Components)

## [Website Demo](https://tesla-car-clone.vercel.app/)

### How To Run The Project:

1- Clone the repo into your local device
2- run 'npm install'
3- run 'npm start'
